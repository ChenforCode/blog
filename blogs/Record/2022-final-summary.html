    # 2022年终总结

    但别在那暴雨时刻，躲在屋檐之下藏身

    # 今年的小总结

    21年自己陷入自我怀疑，纠结的怪圈，让我的一年都过得那么痛苦。庆幸的是今年初及时调整了自己，算是走上了正路。今年年初跨年的时候，自己写下了自己在去年遇见了那么多的困难，但还是和yy两人一起挺过来了，今年的困难似乎不必去年少呢，但是还好自己的大方向是正确的，有人及时指路，及时调整，目前为止，还算是正常。自己回望过去的一年，其实也想不起来自己干了什么，但这一年还是过去了，只是大概记得自己从学校正式走向了公司，从来之前的担忧（害怕自己不能胜任），到现在已经能够独立负责一些业务，自己的成长还是有的，但是还是没有自己预期的快吧。

    # 今年都干了什么？

    2021年12月31日，我们一起在燕园，冬天很冷，但是两个人在一起很暖和。

    1月份：1月1日，其实很多记忆已经记不起来了，只是看着手机的相册能够回忆起来那天在干什么，今天新年的第一天，我俩就在图书馆上自习，直到晚上十一点多才回去（完全理解不了两个傻子在干什么），1月3日，依旧是在图书馆自习（真爱学习呀）。1月7日，晚上在新中关吃了云海肴，菜是真的一般，我现在都可以回想到那个味道哈哈哈，然后还因为筷子上粘了米饭去夹菜嫌不礼貌还吵了一架，气死我了。点了一个烤鱼，一个柠檬鸡（还没学校里的仔鸡饭好吃），一个面包蘸冰激凌（神仙吃法，神仙难吃），一个鸡爪（吃不惯这种无骨鸡爪，还是家里的卤鸡爪好吃），1月10号在家园吃了早饭（早饭其实应该每天都是在家园吃的，偶尔拍一个），今天专利受理了，应该达到毕业要求了，1月11日，好像是浩洲来了，然后还帮他预约入校，但是两个傻子不知道预约入校的核酸是从入校时候起算的48h，不是预约时候的48个小时，所以就被驳回了也没能进来，1月11号的时候应该是准备回家的最后一晚上了，买了泸溪河和鲍师傅的小贝，12号回家。15号吃了海鲜，鸽子汤乱七八糟的广东风味吃不惯，没啥味。20号在家吃火锅，21号妈妈过生日，但是不记得是啥原因了好像也没出去吃饭，在家里吃的蛋糕，继续吃火锅。22号在家闲的没事干把吃火锅剩的牛肉煮了煮成了麻辣烫。23号下雪了，出去逛了逛，我们四个。28号自己在家里炖的羊蝎子（那个时候真的爱做饭），29号在家里煮烩面。30号看见空间里有人发的庐山下雪了，好好看。31号应该是除夕了，不出意外应该是喝多了，哈哈哈哈每次都说过年好好过过瘾，但是每次都喝多（和我妈喝了一瓶白的，一瓶红的），喝到后边，喝到春晚都快结束了，脑子清醒，但已经身体不受控制了，我还专门睡之前去吐了一次，结果还是喝多了。

    2月份：1号，书接上回，今天没拍照，在家里躺了一天，本来今天准备去二舅家吃饭的，奈何喝多了的第二天真的是很难受。到初二也就是2号才缓过来，今天姐夫，叔婶都来了，拍了一张全家福，下午大姐也来了。5号是聚会的日子，来了有十几个把蒹葭里，因为刚喝酒没几天，今天也没喝，下午草草收场去和辉做核酸了。8号那天妈一直咳嗽，咳嗽很多天了，给他熬了个红梨水。11号，13号不出意外，继续吃火锅，17是老家会的日子，去家里赶会了。18号应该是准备要去学校了，想吃鸡蛋饼，俺妈也是第一次做，吃了数不清多少个。19号一起去沙滩上玩了，晚上一起和叔吃了饭。21号应该是最后一晚上了，吃了姐弟俩。22号出发去学校了（虽然有一万个不情愿吧），到现在为止感觉核酸什么的都还挺正常的，只要带着医院的48小时核酸就能进北京了。下午给床铺收拾了一下23号就开始去公司了。

    3月份：1号：这悠长命运中的晨昏，常让我，望远方出神，这是晚上十一点在天作楼下，坐332路在等车的时候想到的，十一点的车要不是末班车，要不就是倒数第二班了，一个人都没有。4号5号都比较常规了，白天在实验室，晚上去学校吃饭，再晚点饿了去买瓶饮料。6号是自己一个人去百讲看了这个杀手不太冷静，8-13号好像一直都没啥写的，和上边一样。18号北京突然下雪了，下了好大的雪，西南门外边路上的草坪都堆满了雪，我记得给我姐发雪的照片他还不信有这么大。20号的时候是和师兄弟一起吃了饭。26号的时候正好是学校里的宜宾美食节，去打卡了一下宜宾燃面（好干而且好辣，对不吃辣的人太不友好了），晚上和yy一起去了前门大街，好久没出去玩了应该是，然后去吃了局气，吃了烤鸭。

    4月份：一直在学校家园吃饭平时，现在翻到这些照片好饿，感觉每一个都好好吃的，学一的牛肉面，家三的仔鸡饭，扣肉饭，菌菇饭，麻辣烫，肠粉还有北冰洋，真的是救了我的老命了。15号的时候是第一次发现了勺园中餐厅（以前其实也知道，但是一直没去打卡过），结果发现里边的菜做的还算精致，19号的时候应该是去五四散步了，这个时候天气已经逐渐转暖了，短袖短裤都出来了。20号继续吃勺园中餐，最爱的西湖牛肉羹，发现和杭州的做法居然是一样的，但是和我印象里家里的做法不太一样，没有家里的做法好吃。21号买了票要出发去武汉了。22号去做了核酸。在实验室把东西都收拾走了，临时和jj打了招呼，在北京的最后一夜。23号早上出发了。yy去送了我，其实我是不喜欢有人去送我的，特别是送我离开，感觉会不舒服。下午cy去武汉站接到了我，带我去住的地方，晚上去了公司，确定了工位，收拾了下桌子，带上了cy给我的钥匙串。24号，买了公司楼下的茶颜和tb一起聊了聊，下午去江边转了转（为数不多的出来玩了）。27号的时候基本已经开始正常工作了，中午尝了尝热干面（这边的芝麻酱跟火锅芝麻酱完全不一样，不好吃）。四月份的最后一天，发了一个朋友圈，算是最近的一个总结吧。

    5月份：五月是火热的一个月，茶颜悦色在办活动，冲了点钱，基本每天都在喝（太热了主要是）。7号海底捞。8号是我的生日，也是yy的生日，俩人一个阳历一个阴历居然能凑到一起，其实生日也很久没正儿八经过过了，晚上收到了yy的蛋糕。9号的时候把家里的家具都擦了擦，用贴纸粘了粘，粘成了蓝色，比原来的大红色好看多了。16号存款第一次破万（惨兮兮靠学校的低保度日），24号关键词：稿件正刊录用。28号，海底捞。

    6月份：6月份前半程的记录好少，自己一直在做公司的事情，也很少出门。25号，yy来找我了，我去汉口站接她，回来的时候都已经很晚了，带她去住的地方收拾了一下，晚上就出来继续买茶颜，吃海底捞。26号算是他在这里呆的完整的一天，白天好热，在商场里一直呆着吃喝看电影也没出门，到了下午带他去了江边看了看，团聚的日子总是短暂的，或许分别才是常态？28号，今天又忙到了很晚，回家的时候已经下起了雨，到家收拾收拾，习惯性的去阳台上站一会，才发现今天的雨好大，从十几楼往前看过去是一片一片的雨幕，这个声音，似乎像是这几个月来的压力在努力的从身体里迸发出来，仔细听着雨声才想起来自己已经好久没有留意身边的东西了。自己似乎总会在某段时间之后有意无意的提醒我自己，可以稍稍放慢些脚步，但是自己又喜欢钻牛角尖，每天总是很晚很晚，只有自己晚上躺在床上，仔细回想今天干了什么之后，才稍稍舒一口气，今天没有白干。每次下雨都会想到这首词，少年听雨歌楼上，红烛昏罗帐。壮年听雨客舟中，江阔云低断雁叫西风。而今听雨僧庐下，鬓已星星也，悲欢离合总无情，一任阶前点滴到天明。不是因为这首词我觉得很伤感无奈。相反我反而从这里看出了一些我很喜欢的点，人呢，每个人都会从少年变成青年壮年再到老年。只要在每个阶段去做自己该做的事情就好了。还有很长很长的路要走，前方的路是江阔云低，前方的路是悲欢离合，但又怎能阻挡你的脚步呢少年！明天还在等着你呢。

    7月份：7月的第一张照片是在5号，稳定发挥，继续吃海底捞，10号，tb的毕业事宜已经基本搞完了，学校的纪念礼物，特别是那个印章好好看，每个人是刻的自己的名字。17号是自己买了一个锅（人生中自己买的第一口锅）煮了一碗泡面吃。20-24号是马哥和粥来的日子，和马哥自从毕业南昌一别，就再也没有见过，他一直在考研，平时也比较少打扰他，这次是考研的结果尘埃落定，难得出来一起嗨皮了吧，粥的话是在北京见过几次，来武汉也是奶茶海底捞乱七八糟一起嗨。24号，最后一顿，吃完大家就各奔东西了。27号给自己的电脑加了两根内存条，32g成就达成。30号晚上可能是太热了，整了两瓶冰啤酒。31号，吃海底捞。

    8月份：6号，公司团建，选了好几百一位的日料自助，奈何我这完全不符合胃口，哈哈哈海胆三文鱼鱼子酱鲍鱼之类的（味道好腥，也没什么特殊的味道，就是咽不下去，因为我对腥味极其敏感），最终从烤大虾和哈根达斯上找补回来一点，算是没亏太多。7号楚山晓。13号公司停电了，从19层走楼梯走了下去，14号发现自己电脑放在桌子上放不稳了，我以为是桌子不平，后来才发现是电池鼓包了，把整个背面撑起来了。14号楚山晓。16号继续停电，18号吃海底捞，19号继续停电。21号去修了电脑，还好电池鼓包属于质量问题保修，省了我1500+，这一段时间好像一直在停电，上午有电，下午停电，然后被迫把工作时间调整到了下午，晚上没有再上班了。22号楚山晓。29号楚山晓。

    9月份：3号楚山晓，10号海底捞（这次好像是自己去的，还整了啤酒喝）。11继续火锅，不过这次是从外卖买的，在家里自己煮了火锅，在家里囤了一箱啤酒。14号和两只傻狗视频喝酒。18号换了个新的鼠标垫，丢掉幻想，准备斗争。22号海底捞。

    10月份：国庆第一天，回家，第二天回老家，第四天和叔聚会，第五天继续聚会，6号回去了。大晚上的到汉口站还得出去做核酸，好惨，下着雨，排着队。因为人太多了没打到车，自己坐地铁回去了，到了华科这里同样下雨叫不到车，自己步行走到家了，走了快一个小时了，好惨。7号楚山晓。9号海底捞。11号在家煮了会面，这个老方城烩面真的挺好吃的，就是22一盒真是傻子才买。我在家里吃一碗也就10块钱。人傻钱多。11号两周年快乐！突然想吃榴莲千层，yy给我买了一个，真的好大没吃完。21号公司两周岁，也是吃了一堆肯德基，一大块蛋糕。22号海底捞。26号楚山晓。30号楚山晓。

    11月份：11月份好像是禁止堂食的一月，到了8号才开放了堂食，和tb俩人好久没正儿八经吃过一顿饭了。13号商场里的松鹤楼面馆开了去尝了尝哈哈哈发现确实很一般。感觉就是酱油汤+挂面，没啥味道，面也不好吃。16号发现了商场里新开的川菜馆，跷脚牛肉真香。21号继续川菜。11月底好像是小区开始封控了，然后公司也封控了，一直在家里居家。30号恢复正常，回来公司了。

    12月份：这最后一个月，我仿佛只过了前半个月。15号好像是在公司吃的最后一顿了？然后就感冒发烧了。我记得是17号周六的时候，坐在这里眼睛一直很不舒服，我一直以为是戴着口罩+眼睛导致的视力下降，结果到了晚上就感觉不太对劲了，量了体温发现确实是发烧了。然后，然后，我22年的最后两周就没了，在家里躺了两周，直到今天我才回来公司，坐在这里赶在今年的末尾把今年是的事情过了一遍。

    还有一些文字记录在了22年的日常感悟里了，这里写的可能比较潦草，但是另外一篇也因为各种各样的原因一直在停滞，最后的一次更新应该是在22号。

    # 还想说点什么呢？

    今年来武汉的原因有点戏剧性吧，来得很急，但是这里可能不太方便细说了，不过能来到这里也算是自己迈出了一大步，其实在寒假里到3-4月份的时候，自己是一直在找暑假实习，奈何自己太菜了，微软和阿里都挂到了最后一面，微软是因为终面算法题没做出来，阿里的话就挂的有点莫名其妙（纯聊天45min然后转手挂了）。其实这个时候的自己是比较迷茫的。还好tb算是给我打了一针强心剂把，直接来武汉了。来这边先是自己独立负责了一个小应用，最终的效果就是做出来的但是由于其他的一些原因还是停滞了。然后就是花了半个月的时间去搞了另外一个应用。然后才算是正式的参与到这边的工作来，刚开始负责的也是一些比较独立的业务，所以自己可以单独拎出来搞，也算是落地了一些模块。然后知道10月份之后，开始正式的接触到了比较核心的业务，自己也能够单独的写了一些页面可见的一些功能。也算是有一丢丢的成就感吧。

    然后另外一件大事就是yy工作终于确定了，从最开始的海投，海笔试，各种算法题轮番轰炸，然后开始海面（中间经历过怀疑人生的阶段，因为很多公司面完就没下文了，以及很多大厂在笔试完约面试也没下文了，要不就是面完一面没下文了），在经历了很长一段时间，才算是拿到了第一个gxb的一个研究所的offer，但是薪资+区位都很差。然后第二个offer就是一个半拉国企半拉私企的那种，薪资比较合适，但是最终还是因为各种负面的评价拒绝了。然后就是讯飞，算是诚意满满，给了很高的薪资和福利待遇但是最终还是因为合肥拒绝掉了。走到这里以为秋招凉了呢。但是接着来了阿里和蚂蚁的offer，很开心，但是没想到这两个offer会到最后变成选择恐惧症哈哈哈。最终的决赛圈就是华子阿里和蚂蚁，华子给的薪资很高很诱人，但就是一直卡着不发offer（不敢完全的allin所以一直也没有太上心）最终在北京和成都选择了成都。至此另外一件大事也完成啦，希望能在蚂蚁里，一个富有挑战性的组里，干出一番有成就的事吧。

    今年（学年）也就是研三了吧，研究生的最后一年，也是学生生涯的最后一年了，在与工作做好各种交接的过程中，一定要注意论文，最后一篇论文了，也是最后一个坎了，一定要认真认真再认真，希望自己能顺利毕业。

    # 明年的期望呢？

    1. 顺利毕业
    2. 和yy在学校里拍非常好看的毕业照（我要去西门拍）
    3. 顺利入职，拿到第一笔工资（非常有纪念意义）
    4. 多回家看看（没有寒暑假了，只有节假日了，能回家就回家）
    5. 都好好的（身体倍棒，能出来旅个游就更好了）

    # 结尾呢，以许嵩的天知道结尾吧，

    信念，就像受潮火种

    有亮不起来的可能

    际遇，一如抓耳挠腮

    也解不出来的方程

    但别在那暴雨时刻

    躲在屋檐之下藏身

    整装前行分秒必争
